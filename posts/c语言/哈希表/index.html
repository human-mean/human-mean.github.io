    哈希表

    February 3, 2026 · 1 min

    哈希表是什么#

    哈希表通过建立键 key 与值 value 之间的映射，实现高效的元素查询，只需要输入一个key就能获得对应值的value。　

    哈希表与链表、数组的时间复杂度比较：

    数组链表哈希表
    查找元素O(n)O(n)O(1)
    添加元素O(1)O(1)O(1)
    删除元素O(n)O(n)O(1)

    哈希函数#

    一个把数据转换成数组下标的规则，只有两个要求：

    1. 同样的输入，永远得到同样的输出
    2. 输出范围在 0 到 size-1 之间

    输入一个key，我们可以通过哈希函数获得key值对应键值的存储位置。

    哈希函数的计算步骤：

    1.通过一个哈希算法hash()得到哈希值。 2.将哈希值的数组长度capacity取模，从而获取对应的index

    index = hash(key) % capacity

    因为哈希函数可能算出很大的数，但数组下标是有限的。取模可以把任意大的数压缩到 0 到 capacity-1 的范围内。

    哈希冲突与扩容#

    哈希函数的作用是将所有 key 构成的输入空间映射到数组所有索引构成的输出空间，而输入空间往往远大于输出空间。因此理论上一定存在多个输入对应相同输出的情况。

    比如：

    7 %  10  =  7
17 % 10  =  7

    哈希表容量越大，输出相同的可能性就越低，通过扩容哈希表可以减少哈希冲突
